# Команды

- Установка библиотек

```
npm i
```

- Запуск

```
npm run build && npm run preview
```

Проект запустится на [http://localhost:4173/](http://localhost:4173/)

- Запуск в dev-режиме

```
npm run dev
```

Проект запустится на [http://localhost:5173/](http://localhost:5173/)

- Форматирование кода

```
npm run format
```

- Линтер

```
npm run lint
```

# Requirements

- Node v18+
